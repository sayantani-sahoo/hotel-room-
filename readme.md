# Sahoo Royals Hotel Website

A modern, responsive hotel booking website built with HTML, CSS, and JavaScript. Features smooth animations, interactive booking system, and mobile-friendly design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: CSS animations for navbar, hero section, and background image
- **Interactive Booking System**: Modal-based booking form with validation
- **Room Gallery**: Showcase of different room types with pricing
- **Food Menu**: Display of dining options with images and prices
- **Contact Information**: Complete contact details and location
- **Mobile Navigation**: Hamburger menu for mobile devices

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins)

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/sayantani-sahoo/hotel-room-.git
   ```

2. Navigate to the project directory:
   ```bash
   cd hotel-room-
   ```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local server (recommended for better performance)

## Usage

- **Navigation**: Use the navbar to jump to different sections
- **Booking**: Click "Book Now" buttons or the navbar link to open the booking modal
- **Mobile**: Tap the hamburger menu for mobile navigation
- **Form Validation**: The booking form includes date validation and required fields

## Features in Detail

### Animations
- Navbar slides down on page load
- Hero content slides up from bottom
- Background image zooms in smoothly
- Room cards slide in with staggered delays
- Mobile menu transitions smoothly

### Booking System
- Room type selection
- Date picker with validation
- Guest count selection
- Payment method options
- Special requests textarea
- Form validation and confirmation

### Responsive Design
- Desktop: Full navigation and grid layouts
- Tablet: Adjusted spacing and font sizes
- Mobile: Hamburger menu and stacked layouts
